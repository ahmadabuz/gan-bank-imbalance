# gan-bank-imbalance

-i used colab to write the code , so for a smooth experience it's recommended to use colab

-when using colab make sure you upload the dataset csv file : bank-full_sp.csv before running the code 

-you will have to wait a couple of minutes to get all results 

-or to get a faster performance:

1)after navigating to colab , click on runtime

2)then click on Change runtime type, under Hardware accelerator choose T4 GPU
